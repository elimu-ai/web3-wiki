# github-contributors

> Calculation of impact percentages for GitHub repos

See [`../../DRIP_LIST_FUNDING_SPLITS.md#content`](../../DRIP_LIST_FUNDING_SPLITS.md)

## Install Dependencies

```bash
npm install
```

## Run the TypeScript Compiler

```bash
npx tsc
```

## Run the JavaScripts

```bash
node calculate-percentages.js
```
